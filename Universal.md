# 环球影城（Universal Studios）

好莱坞环球影城，被称之为成年人的迪斯尼乐园，把唯一的、原汁原味的好莱坞幕后体验与崭新的娱乐项目相结合，从台前到幕后，从现实到虚幻，融合成全新的、互动的电影制片之旅。这次我将结合自身的经历，总结为一篇游玩攻略供大家参考，少走一点弯路，多一分快乐。

<!-- more -->

# 门票（Tickets）
首先说说门票。门票的形式为通票，也就是说在门票所限日期内进入园区之后，任何一个游玩项目都是无需额外付费的。门票的种类主要有单日通票、前排通票以及VIP通票。单日通票票价为100美元，前排通票和VIP通票分别为200美元和300美元左右，这两种高端门票都有无需排队的特权，而VIP门票更是有随行陪护。我们这次所买的门票全是单日通票，虽然没有排队的特权，但是之后我会为大家介绍一个小技巧，能够使用普通票也能大幅缩短排队时间。门票可以在官网订购，打印下来作为入园凭证。

# 步行街 (Citywalk)
环球影城早晨9点开园，暑假正值旺季，为了尽量玩到所有项目，我们也赶了个早，九点多就到了，下车之后就是 Citywalk 的门口（类似Disney，在未到公园正门但可以逛街的地方），这条街通往正门，集餐厅，纪念品店和影院的一条街。因为不在园区内，所以都是要付费的，不在门票费用之内。
