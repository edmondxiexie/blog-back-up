# 环球影城（Universal Studios）

<div  align="center">    
    <img src="https://c6.staticflickr.com/6/5328/30592336581_9a6ffa1861_k.jpg" width = "1000" alt="好莱坞环球影城" align=center title="" />
</div>

好莱坞环球影城，被称之为成年人的迪斯尼乐园，把唯一的、原汁原味的好莱坞幕后体验与崭新的娱乐项目相结合，从台前到幕后，从现实到虚幻，融合成全新的、互动的电影制片之旅。这次我将结合自身的经历，总结为一篇游玩攻略供大家参考，少走一点弯路，多一分快乐。

<!-- more -->

# 门票（Tickets）
首先说说门票。门票的形式为通票，也就是说在门票所限日期内进入园区之后，任何一个游玩项目都是无需额外付费的。门票的种类主要有单日通票、前排通票以及VIP通票。单日通票票价为100美元，前排通票和VIP通票分别为200美元和300美元左右，这两种高端门票都有无需排队的特权，而VIP门票更是有随行陪护。我们这次所买的门票全是单日通票，虽然没有排队的特权，但是之后我会为大家介绍一个小技巧，能够使用普通票也能大幅缩短排队时间。门票可以在官网订购，打印下来作为入园凭证。

# 步行街 (Citywalk)
环球影城早晨9点开园，暑假正值旺季，为了尽量玩到所有项目，我们也赶了个早，九点多就到了，下车之后就是 Citywalk 的门口（类似Disney，在未到公园正门但可以逛街的地方），这条街通往正门，集餐厅，纪念品店和影院的一条街。因为不在园区内，所以都是要付费的，不在门票费用之内。

如果不在街上过多逗留，沿路步行约10分钟就可看到标志的旋转地球仪了，旁边是售票，换票点和正门。进园的时候记得拿一张园区的地图，地图上详细地标出了各游乐设施的位置、简介，同时还有各个表演和剧场的场次时间。

环球影城作为一个电影主题乐园，随处都能感受到电影的氛围，一进入大门，迎面而来的就是一组描写片场拍摄工作的雕塑，围绕在喷泉四周有用手取景的导演，坐在摇臂上的摄像，还有打板的场记。值得一提的是这尊举着场记板的人像雕塑，可是由真人扮演的，如果有人站过去合影，他会趁你不注意猛地一打板，吓你一跳，然后又恢复到原来的姿势。

然而，千万不要被这些雕塑吸引太久，因为园区内游客非常多，一些重点项目的排队时间基本在40分钟左右，新开放的哈利波特虚拟过山车甚至要70分钟，所以最有效的游园方式，不是按地图顺序绕一圈玩，而是在排队低谷期把重点项目先玩了，其他的慢慢转。而第一个要去的项目，就是环球影城的重中之重、镇园之宝、非去不可不然不算是到过环球影城的影城之旅(Studio Tour)。

# 影城之旅 (Studio Tour)
所谓的 Studio Tour， 就是大家坐着小火车，参观Universal 拍片的片场和摄影棚，每个时间段发车的语言服务都不同，我们赶到的时候正巧是一辆普通话解说的车，也是十分幸运，导游是个台湾腔的妹子，非常热情，和国内温吞水导游完全不同。导游妹妹坐在手节车厢，每节车厢都装有电视和音响，所以所有人都能看清她的丰富的面部表情。在上车前要记得拿3D眼镜，有一处整个小火车会进入一个大棚，四周都变成屏幕，上演一出追车戏份，如同身临《速度与激情》的世界，有真实的爆炸，真实的洪水，真实的颠簸感，是从未有过的体验。
